ATENÇÃO: A PASTA .github É OBRIGATÓRIA

O GitHub somente executa a atualização quando este arquivo estiver no caminho:

.github/workflows/atualizar-radar.yml

Não envie apenas os arquivos que aparecem na raiz.

FORMA MAIS SEGURA DE ENVIAR PELO SITE DO GITHUB

1. Extraia o ZIP.
2. No repositório Gmaquina1/radar, abra Add file > Upload files.
3. Arraste primeiro a pasta .github inteira para a tela do GitHub.
4. Confirme que aparece:
   .github/workflows/atualizar-radar.yml
5. Depois arraste todos os demais arquivos.
6. Confirme o commit na branch main.
7. Abra Actions > Atualizar Radar e novos lotes.
8. Clique em Run workflow caso a execução não comece automaticamente.

COMO SABER SE FUNCIONOU

Abra status_atualizacao.txt no repositório.

A data da base de lotes deve ser a data da nova execução.
O total pode permanecer igual quando entram e saem quantidades semelhantes.
Por isso confira também:
- Lotes novos detectados
- Lotes removidos ou encerrados
